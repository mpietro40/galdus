Servizi di memorizzazione e backup online

Servizio di salvataggio

Un servizio di salvataggio è, appunto, un servizio online che permettere di salvare qualsiasi file, password, ecc.

Servizio di sincronizzazione

Un servizio di sincronizzazione è un servizio online e non che permettere di sincronizzare qualsiasi file e poterlo visualizzare online o in qualsiasi dispositivo con tale applicativo.

Servizio di condivisione file in cloud

Un servizio di condivisione file in cloud è un servizione che permettere di condividere e salvare qualsiasi file all'interno di un cloud.
