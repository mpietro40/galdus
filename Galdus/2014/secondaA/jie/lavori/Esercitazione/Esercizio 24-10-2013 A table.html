Webville Guide to HTML

Webville Guida a HTML

In this handy guide, we've boiled down writing well-formed HTML pages into a common sense set of guidelines. Check them out:

In questa pratica guida, noi abbiamo bollito giu' la scrittura di pagine HTML ben formati in un senso comune set di linee guida. Dateci un'occhiata:

Always begin with the < doctype >.

Iniziare sempre con la < doctype >.

Always start each page with a doctype. This will get you off on the right foot with browsers, and with the validator too.

Iniziare sempre con la . Iniziare sempre ogni pagina con un doctype. Questo ti porterà fuori il piede giusto con i browser e con il validatore troppo.

Use < ! doctype html > at all times, unless you really are writing HTML 4.01 or XHTML.

Utilizzare < ! DOCTYPE html > in ogni momento, a meno che realmente sta scrivendo HTML 4.01 o XHTML.

The < html > element: don't leave home without it.

Il < html > elemento: Non uscire di casa senza di essa.

Following the doctype, the < html > element must always be the top, or root, element of your web page. So, after the doctype, the < html > tag will start your page and the < / html > tag should end it, with everything else in your page nested inside.

Il < html > elemento: non uscire di casa senza itFollowing il doctype, il < html > elemento deve essere sempre la parte superiore, o radice, elemento della tua pagina web. Così, dopo il doctype, il < html > tag inizierà la pagina e il tag dovrebbe finirla, con tutto il resto della tua pagina nidificato all'interno.

Remember to use both your < head > and your < body > for better HTML.

Ricordate di usare sia la vostra < head > e il tuo < body > per una migliore HTML.

Only the < head > and < body > elements can go directly inside your < html > element. This means that every other element must go either inside the < head > or the < body > element. No exceptions!

Solo l'< head > e < body > elementi possono andare direttamente all'interno del vostro < html > elemento. Questo significa che ogni altro elemento deve andare sia all'interno del < head > o il < body > elemento. Nessun eccezioni!

Feed your < head > the right character encoding.

Alimentare il vostro < head > la codifica dei caratteri a destra.

Include a < meta charset="utf-8" > tag in your < head >. The browser will thank you, and so will your users when they're reading comments on your blog from users around the world.

Includere un < meta charset = "utf-8" > tag nel < head >. Il browser vi ringrazierà, e così anche gli utenti quando stanno leggendo commenti sul tuo blog da utenti di tutto il mondo.
