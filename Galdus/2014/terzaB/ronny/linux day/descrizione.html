Word Press

WordPress è un CMS (Content Management System) open-source sviluppato in PHP.
Utilizzando WordPress è possibile creare blog o siti web (anche piuttosto complessi) senza scrivere nemmeno una riga di codice. E', inoltre, possibile personalizzare l'aspetto grafico di WordPress installando uno dei tanti template gratuiti oppure creandone uno personalizzato (in quest'ultimo caso, però, sono necessarie delle conoscenze di HTML e CSS).
Una volta installato WordPress sarà possibile gestire i contenuti del sito in modo molto semplice, attraverso una comoda interfaccia di amministrazione dalla quale sarà possibile - tra le altre innumerevoli cose - scrivere nuovi contenuti o modificare quelli già inseriti.
Per chi fosse interessato, su Mr.Webmaster è disponibile una guida gratuita che spiega come installare, configurare ed utilizzare WordPress per creare il proprio sito web.
