Un server proxy è un computer che funge da intermediario tra il computer dell'utente e Internet. Questo server può essere utilizzato per registrare informazioni sull'utilizzo di Internet e per bloccare l'accesso a un sito Web. Il firewall del server proxy blocca alcuni siti o pagine Web per diverse ragioni. Il firewall potrebbe pertanto impedire il download di Java Runtime Environment (JRE) o l'esecuzione di alcune applet Java.
