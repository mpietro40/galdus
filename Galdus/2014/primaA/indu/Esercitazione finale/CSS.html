CSS

Il linguaggio CSS permette di impostare tutti gli aspetti di visualizzazione di una pagina web, quali allineamento, layout (disposizione dei contenuti rispetto allo schermo), formattazione del testo e colori. Esso è nato dalla necessità di separare il contenuto HTML della pagina dalla grafica e dalla visualizzazione. Le regole per la buona stesura di un codice CSS sono state dettate dal W3C.
