Codice di esempio di una tabella html

Questo codice deve essere inserito nella pagina html all'interno del body

INIZIO CODICE

<table>
<tr>
<th>Titolo 1</th>
<th>Titolo 2</th>
<th>Titolo 3</th>
<th>Titolo 4</th>
</tr>
<tr>
<td>Cella 1</td>
<td>Cella 2</td>
<td>Cella 3</td>
<td>Cella 4</td>
</tr>
</table>

FINE CODICE

nel css inserire le isctruzioni adeguate ad esempio

table, td, th
{
border:1px solid green;
}
th
{
background-color:green;
color:white;
}

Risultato

Titolo 1 Titolo 2 Titolo 3 Titolo 4
Cella 1 Cella 2 Cella 3 Cella 4
Cella 1 Cella 2 Cella 3 Cella 4
