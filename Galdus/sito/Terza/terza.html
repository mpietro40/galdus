Passi per realizzare il sito della tesina

In questa pagina sono inserite le informazioni per realizzare il mini sito della classe terza:
  1. Pensare a cosa si vuole inserire (il tema della tesina)
  2. Creare una pagina html base
  3. Creare una struttura con cartelle per immagini, css, javascript ecc
  4. Inserire i riferimenti al CSS nella sezione head della pagina HTML
    <link rel="stylesheet" href="css/style.css" type="text/css" />
  5. Creare una file .css con alcuni stili CSS
  6. Impaginare i contenuti delle pagine con l'inserimento dei <div> per passare da Pagina senza DIV a Pagina con DIV alla Pagina con DIV e CSS in linea
  7. Creare un menu un possibile menu e con i CSS Menu dinamico
  8. Creare animazioni o grafica del menu
  9. Inserire animazioni javascript (slideshow)
  10. Testare le Pagine
  11. Inserire una tabella Tabella
  12. Inserire un menu' animato in css
