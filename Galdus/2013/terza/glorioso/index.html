Java e Java script
Differenze tra Java e Java Script: Il linguaggio di programmazione JavaScript sviluppato da Netscape, Inc., non fa parte della piattaforma Java. JavaScript non è in grado di creare applet o applicazioni standalone. Nella sua forma attuale più diffusa, JavaScript è presente in documenti HTML ed è in grado di garantire alle pagine Web livelli di interattività superiori rispetto a quelli ottenuti con il semplice linguaggio HTML. Di seguito sono elencate le principali differenze tra Java e JavaScript. •Java è un linguaggio di programmazione orientato agli oggetti, mentre JavaScript è un linguaggio di scripting orientato agli oggetti. •Java crea applicazioni eseguibili su browser o macchine virtuali, mentre il codice JavaScript è eseguibile unicamente su browser. •Il codice Java deve essere compilato, mentre il codice JavaScript si presenta sotto forma di testo. •Java e JavaScript richiedono plugin diversi.
