Esempio di un mini sito web per la gestione di una biblioteca

Creato dai ragazzi della 3 Elettronica GALDUS nel a.s. 2012-2013

In queste pagine e' possibile vedere il lavoro fatto dalla classe divisa in gruppi per realizzare la loro prima web App in php con un motore database sqlite che consente di effettuare le principali operazioni di una normale biblioteca. Il lavoro ha scopo didattico e non e' pronto per essere utilizzato in produzione mancando infatti tutta la parte di analisi e gestione della sicurezza, degli accessi e consapevoli che il DB sqlite mal gestisce la multi utenza.
